# ================================================
# Installation Instructions
# ================================================
#
# Install all dependencies:
#   pip install -r requirements.txt
#
# For OCR support, also install Tesseract:
#   Windows: Download from https://github.com/UB-Mannheim/tesseract/wiki
#            Default install path: C:\Program Files\Tesseract-OCR
#   Linux:   sudo apt-get install tesseract-ocr
#   macOS:   brew install tesseract
#
# For PDF OCR with pdf2image, also install Poppler:
#   Windows: Download from https://github.com/osber/poppler-windows/releases
#            Add bin folder to PATH
#   Linux:   sudo apt-get install poppler-utils
#   macOS:   brew install poppler
#
# Alternative: Use PyMuPDF instead (no Poppler needed):
#   pip install PyMuPDF
#
# Minimal installation (core features only):
#   pip install requests PyPDF2 openpyxl Pillow python-docx pyyaml
